# gfb
gfb is a python framework that exposes a simple api enabling programmers to create music-based applications.

<hr/>

## API

### Note: (class)


### DiatonicKey: (class)
### ChromaticKey: (class)


### Tuning: (class)

```
Tuning()
```


### Arguments

* -h --help  

* -f --frets [int] 
* -v --verbose [int]
